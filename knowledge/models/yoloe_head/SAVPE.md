# SAVPE (空间感知视觉提示词嵌入模块)

`SAVPE`（Spatial-Aware Visual Prompt Embedding）是 YOLOE 预测流中用于对输入视觉提示（Visual Prompts，VP）执行空间位置编码及特征增强的辅助投影头。

---

## 1. 设计初衷与位置

在开放词表（Open-vocabulary）或交互式（Interactive）分割场景中，用户的 Prompt（如一个点、一个框或特定类别的视觉切片）必须能够跟主干网络抽取的特征图进行跨模态空间对齐。
`SAVPE` 作为这套交互机制的核心连接桥梁：
- 接收来自主干网络多级提取特征 `x`。
- 接收初始的视觉提示向量 `vp`。
- 在空间维进行特征的多尺度增强与插值投影，输出高维、对齐的视觉提示词嵌入矩阵，为后续的分类和对比学习提供精准的交互引导。

---

### 构造函数

```python
def __init__(self, ch, c3, embed):
```

| 参数 | 类型 | 描述 |
| :--- | :--- | :--- |

| `embed` | `int` | 输出的高维视觉嵌入向量通道维度（通常为 512）。 |

---

## 3. 结构特征与推理对齐

`SAVPE` 的内部拓扑体现了高度精妙的多尺度上采样与拼接机制：
- **`self.cv1` 与 `self.cv2`（多尺度增强路径）**：
  构建了包含多级 `Upsample` 与双重 `Conv` 运算的并行 ModuleList，用以将不同尺度（Scale 0, Scale 1, Scale 2）的输入特征自动对齐到统一的最大分辨率下。
- **空间融合卷积**：
  融合后的特征随后经过 `self.cv3`（$1 \times 1$ 特征投影层）与 `self.cv4`（$3 \times 3$ 空间密集卷积）完成物理维度的进一步归一化。
- **权重对齐保护**：
  在官方的 `yoloe-26s-seg-pf`（Prompt-Free，免提示词变种）运行时，由于它是免交互运行，`SAVPE` 的实际特征聚合前向结果将在权重折叠后被精简或冻结，其 forward 方法输出的占位矩阵仅用作尺寸对其，不破坏大主干对于各类交互式视觉提示的兼容潜能。
